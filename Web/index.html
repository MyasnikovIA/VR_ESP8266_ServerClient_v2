Соответствие осей:

• Pitch (X) - Наклон вперед/назад (кивок головой) → rotateX()

• Roll (Y) - Наклон к плечам влево/вправо → rotateZ()

• Yaw (Z) - Поворот головы влево/вправо → rotateY()

Порядок вращения: Yaw → Pitch → Roll

Ожидание подключения...

Инструкция:

1. Убедитесь, что ESP8266 включена и подключена к WiFi

2. Нажмите "Подключиться" для установки соединения

3. Двигайте датчик MPU6050 для изменения положения VR-головы

4. Используйте "Сбросить углы" для возврата в исходное положение
